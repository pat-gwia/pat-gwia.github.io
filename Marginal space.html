Boston's Mini Theatre

Boston City Centre, Lincolshire, UK

Home Portfolio About

The marginal space project looks to create a new point of interest in Boston's Market square, Lincolnshire. The project creates a hidden miniature theatre within an archway.

The project uses a lightweight aluminium frame, which houses the recycled plastic seating. Once the seating has been removed the stage opens up using an electrical motor and rails. While not in use the recycled plastic seating is stored within the arch, creating a interesting coloured facade.

When a performance takes place the plastic seating, attached using magnets, is removed from the frame and used as seating for the viewers.
